This Competitor Management System is a Java application that allows you to manage a list of competitors. It provides features to add, edit, delete competitors, view details, and generate reports.


## Table of Contents

- [Features](#features)
- [Getting Started](#getting-started)
  - [Prerequisites](#prerequisites)
  - [Installation](#installation)
- [Usage](#usage)

## Features

- Add new competitors with detailed information.
- Edit existing competitor details.
- Delete competitors from the list.
- View competitor details, sorted by ID or name.
- Update competitor scores.
- Generate a comprehensive final report with statistics.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) installed on your machine.
- An Integrated Development Environment (IDE) such as IntelliJ IDEA or Eclipse.

### Installation

1.Open the project in your preferred IDE.


#Usage

1.Run the GUIDemo class to launch the Competitor Management System GUI.
2.Use the GUI to interact with the Competitor Management System.
